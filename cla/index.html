Contributor License Agreements

Like many open source organizations, Relaycorp requires contributors to sign a Contributor License Agreement (CLA) in order to protect the Relaynet project and related software from disputes and license violations.

CLAs are managed by CLA Assistant, and you will be prompted to sign the CLA when you submit your first Pull Request to the project.

If you implemented the Pull Request on behalf of an organization, you should sign the Relaycorp Entity Contributor Assignment Agreement. If you are an individual contributor, please sign the Relaycorp Individual Contributor Assignment Agreement.
