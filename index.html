About Relaycorp

We’re incubating the following technologies to pave the way for solving some of humanity’s most pressing and neglected communication challenges, starting with Internet blackouts:

Awala

A new computer network where apps can communicate with and without the Internet. It also offers end-to-end encryption and spam protection.

Learn more about Awala

Letro

An Awala-powered email alternative, so it works with and without the Internet, supports end-to-end encryption, and is spam-resistant.

Learn more about Letro

VeraId

An offline authentication protocol that allows data to be securely attributed to domain names (e.g. example.com) and users (e.g. alice@example.com). VeraId powers Letro.

Learn more about VeraId

Despacito

A rate limiting protocol that mitigates DDoS attacks. It’s based on the research we did for the DDoS Report.

Learn more about Despacito
